# 序言

## 有什么

这是一个为所有计算机爱好者编写的一个wiki或者说是教程，这个网站主要包括一些~~详细~~一点也不详细的实践，理论和Q&A。有文字也可能会有图片，如果我想可能会有实操视频但大概率会直接超链接到外部或内嵌视频组件。 

## 为什么要做

1. 方便我与其他可能遇到相同问题的人查阅
2. 方便给一些不懂的人直接甩一个现成的教程
3. 避免反复造轮子
4. ~~练手~~

## 怎么用
左边有文章栏，右边有段落栏，只要不是连电子产品都不会用的应该都知道。

## 怎么跟着学
我的建议是：
1. 先读实践，看到不了解的概念或缩写点进去查看理论或者速查表，了解基本步骤与可能需要注意的点
2. 再次跟着教程实践，遇到问题先检查步骤，然后看Q&A,然后搜索，然后问大模型，然后问群友，试了一遍完全无法解决那我也无能为力了。

## 在这一切之前还需要知道和准备什么
1. 学会尊重别人，承认自己的无知与不足，不要学一点皮毛就在外面显摆
2. 把[如何提问](https://lug.ustc.edu.cn/wiki/doc/howtoask/)和[提问的智慧](https://lug.ustc.edu.cn/wiki/doc/smart-questions/)通读3遍确保自己完全理解了这篇这篇文章，并在提问时运用并遵守这些技巧与规则
3. 不用翻译软件，使用英文进入[CGTN](https://www.cgtn.com/)能够看懂网站上的标题，并在随机一篇文章中了解大意
4. 一个智商和理解能力在线的脑子
5. 不要认为别人给你东西是理所应当的，记住，别人给你东西是情分，不给你东西是本分，没有人会为了一个鸡毛蒜皮的小事情浪费自己过多的时间
